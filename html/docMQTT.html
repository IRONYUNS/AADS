MQTT in MATLAB

Create connection to a broker for communication using MQTT protocol

Syntax

M = mqtt(BrokerAddress)

M = mqtt(_,Name,Value)

Description

M = mqtt(BrokerAddress) create a new MQTT connection to a broker at BrokerAddress

M = mqtt(_,Name,Value) uses additional options specified by one or more Name, Value pair arguments

Examples

MQTT Connection

myMQTT=mqtt('tcp://broker.hivemq.com')

myMQTT =


Mqtt with properties:

    BrokerAddress: 'tcp://broker.hivemq.com'
             Port: 1883
         ClientID: ""
          Timeout: 5
        Connected: 1
        KeepAlive: 60

MQTT Connection with ClientID myClient at Port 1883

myMQTT=mqtt('tcp://broker.hivemq.com','ClientID','myClient','Port',1883)

myMQTT =

Mqtt with properties:

    BrokerAddress: 'tcp://broker.hivemq.com'
             Port: 1883
         ClientID: 'myClient'
          Timeout: 5
        Connected: 1
        KeepAlive: 60

MQTT Connection with SSL

myMQTT = mqtt('ssl://mosquitto.org','Port',8884, ...
    'CAFile', 'C:\mosquitto.org.pem',...
    'ClientCertificate', 'C:\client.crt',...
    'ClientKeyFile', 'C:\client.key');

MQTT Connection with WSS

myMQTT=mqtt('wss://mqtt.thingspeak.com','Port',443)

Input Arguments

BrokerAddress - Host name or IP address of the MQTT broker, specified as a string or character array including connection protocol. Supported protocols include TCP, WS, SSL, and WSS

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN .

Use Name,Value arguments to specify parameter initial values and additional properties of mqtt connection during creation.

Note: All mqtt connections are made as a clean session, where the broker does not remember any previous subscriptions by the client.

'Port'

Socket port number to use when connecting to the MQTT broker, specified as a comma-separated pair consisting of 'Port" and the port number.

Example: myMQTT=mqtt('tcp://mqtt.thingspeak.com', 'Port',1883)

'Username'

Username to use when connecting to the MQTT broker, specified as a comma-separated pair consisting of 'Username' and a character vector. If the Username parameter is not supplied, the connection is made without a user name.

Example: myMQTT= mqtt('tcp://mqtt.thingspeak.com','Username','myName')

'Password'

Password to use when connecting to MQTT broker, specified as a comma-separated pair consisting of 'Password' and a character vector. If the Password parameter is not supplied, the connection is made without a password.

Example: myMQTT= mqtt('tcp://mqtt.thingspeak.com','Username','myName','Password','myPWD')

'ClientID'

A MQTT identifier that identifies the client to the broker, specified as a comma-separated pair consisting of 'ClientID' and a character vector. The broker uses this ID to identify the client and the current state of the client.

Example: myMQTT = mqtt('tcp://mqtt.thingspeak.com','ClientID','NameofClient');

'Timeout'

Waiting time in seconds to complete a connection, specified as 'Timeout' and a numeric value.

Example: myMQTT = mqtt('tcp://mqtt.thingspeak.com,'Timeout',5);

'KeepAlive'

Longest time connection will persist without sending a message, specified as a comma-separated pair consisting of 'KeepAlive' and and a positive integer in seconds.

Example: myMQTT = mqtt('tcp://mqtt.thingspeak.com,'KeepAlive',60);

'CAFile'

Path to Root Certificate specified as 'CAFile' and a path and file name. Learn more about Secure Connections.

Example: myMQTT=mqtt('wss://mosquito.org','Port', 883,'CAFile','C:\CertFiles\mosquitto.org.crt')

NOTE: You can generally obtain the CA file from the broker directly. For example, you can find information on how to obtain client certification information from mosquitto.org here.

'ClientCertificate'

Path to client certificate, specified as'ClientCertificate' and a file path and location. Learn more about Secure Connections.

Example: myMQTT=mqtt('wss://mosquito.org','Port',883,'ClientCertificate', 'C:\CertFiles\client.pem')

'ClientKeyFile'

Client Key file 'ClientKeyFile' and a file location. Learn more about Secure Connections.

Example: myMQTT=mqtt('wss://mosquito.org','Port',883,'ClientCertificate','C:\CertFiles\client.pem','ClientKeyFile', 'C:\CertFiles\client.key')

'SSLPassword'

Password for SSL certificate 'SSLPassword' and a password. Learn more about Secure Connections.

Example: myMQTT=mqtt('wss://mosquito.org','Port', 883,'ClientCertificate','C:\CertFiles\client.pem','ClientKeyFile','C:\CertFiles\client.key','SSLPassword','12345')

Object Functions

mqtt object functions include:

subscribe

Subscribe to a topic on MQTT broker

publish

Publish a message to MQTT broker

unsubscribeAll

Unsubscribe from all topics on MQTT broker

Output Arguments

mqtt object. After creating the mqtt object, you can use dot notation to access or change its property values.

myMQTT=mqtt('tcp://broker.hivemq.com')

 Mqtt with properties:

   BrokerAddress: 'tcp://broker.hivemq.com'
            Port: 1883
        ClientID: ""
         Timeout: 5
       Connected: 1
       KeepAlive: 60
