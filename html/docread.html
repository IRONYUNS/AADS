read

Read messages from MQTT subscription

Contents

  • Syntax
  • Description
  • Examples
  • Input Arguments
  • Output Arguments
  • See Also

Syntax

message=read(s)

[message,Time]=read(s)

Description

message=read(s) reads the most recent message from MQTT subscription s

[message,time]=read(s) additionally returns the time that the message was received.

Examples

Read Latest Message in Subscription

%Create MQTT connection.
myMQTT=mqtt('tcp://broker.hivemq.com');

%Add Subscription
mySub=subscribe(myMQTT,'IoTtopic');

%Read Subscription Message
message=read(mySub)

message=

      "My Message"

%Read Message and Time Message Received
[message,time]=read(mySub);

message =

    "hello world"


time =

    datetime

    11-Aug-2017 14:37:45

%Only the latest message is displayed

Input Arguments

s - MQTT subscription, specified as the variable name representing the subscription.

Output Arguments

message - message returned as a string.

time - time message was received, returned as a datetime.

See Also

|mqtt | publish | subscribe | readall | unsubscribe |
